## 最近特色

### 新年祝福 - V1.4.3
- 订阅新年祝福短语合集，不再为发祝福消息发愁啦~

### 键盘皮肤 - V1.4.0
- 重新调整了键盘的样式，增加了第一款键盘皮肤 - 「初心」
- 订阅后也可以买断进阶版
- 优化键盘功能操作

### 布局变化 - V1.3.0
- 重新调整ToMemo的功能界面，增加三个tab栏——「Memo」、「短时记忆」、「设置」
- 针对Memo和短时记忆增加多选功能：
  - Memo：移动、复制、删除
  - 短时记忆：合并、删除
- 优化记录的操作方式，操作按钮都放到页面上方
