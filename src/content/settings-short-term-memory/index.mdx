# 短时记忆设置

## 开启短时记忆功能

- 在应用使用过程中，监听到剪切板的变化，并存入短时记忆
- 在键盘使用过程中，将监听到的剪切板的内容存入短时记忆（需要在键盘设置中自动读取剪切板）
- 通过触发快捷指令，将剪切板的内容迅速保存入短时记忆
